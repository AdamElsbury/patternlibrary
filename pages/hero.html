Hero section

Element has a pre-configured hero section. A hero section is the main section at the top of a web page which contains the navigation bar, business or service name, a short description and usually (but not always) a call to action button.

Basic usage

HTML markup

Start with a .hero container, add the navigation bar, add a new row for the header elements and call to action and voila! See below for full HTML code.

<div class="hero"> <div class="nav-container"> <ul class="nav-links item-2"> <li class="nav-item"><a href="#">LINK 1</a></li> <li class="nav-item"><a href="#">LINK 2</a></li> <li class="nav-item"><a href="#">LINK 3</a></li> </ul>
<div class="container"> <div class="grid-row center-content hero-row"> <div class="grid-col"> <h1>Your business name</h1> <h2>Your business description</h2> <button class="btn-primary">CALL TO ACTION</button> </div> </div> </div> </div>

Adjust height of the hero section

In order to adjust the height of the hero section change the height selector in the CSS rule for .hero-row. It is set to 300px as default.

.hero-row { height: 300px; }

Change alignment of content

The hero row is set to have its content vertically and horizontally aligned. It does this through the .center-content class. Additionally the text is being centered within the container using text-align: center from .hero.

<div class="nav-container"></div>
.hero-row { text-align: center; }
