For developers

Element vanilla library

The library provides front end engineers a collection of reusable web elements to plug into your own interfaces. All components are comprised of HTML, JS and SCSS partials to allow you to easily copy the code into your project. Simply install through npm or download the files and insert them into your project manually.

Installation

If you are using npm:

$ npm install --save element

If you want to download the project:

Element CSS

This version uses standard CSS, HTML and JavaScript. This option is better for beginners.

Element SCSS

This version uses SCSS which needs a SASS compiler to work. A little more tricky for beginners!

Dependencies

Element uses a few dependencies to work. Please ensure they are installed else this design system will not work.

node-sass

Node-sass is a SASS compiler that is used to read this projects SCSS files. This dependency is only needed for the SCSS version of the project.

$ npm install --save node-sass FULL DOCUMENTATION

jQuery

jQuery is a JavaScript library used all over the web. jQuery is used in this project for things like the modal component, navigation bar and some animations. Install either with npm or adding script tags to the top of your HTML page.

If you are using npm:

$ npm install --save jquery

If you are using script tags:

<script type="text/javascript" src="https://code.jquery.com/jquery-3.3.1.min.js"></script> FULL DOCUMENTATION

Font Awesome

Element uses the free open source Font Awesome icon library. Installation for this is simple, include the following stylesheet link in your HTML document.

Include in head:

<link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.0.12/css/all.css">
