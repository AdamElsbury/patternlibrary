Responsive text

Responsive text

Element uses the CSS calc property to calculate the size of the text on the webpage depending on the viewport width and a range of two px sizes (minimum & maximum).

The text will adjust automatically on the page. Adjust your browser width to see the text resize in real time. The sizing applies to all header elements as well as paragraphs.

<h1>Hello world!</h1> <h2>Hello world!</h2> <h3>Hello world!</h3> <p>Hello world!</p>

If you wish to change the size of the text you can adjust the range so the calculation makes the text bigger or smaller, like adding a media query for mobile viewports.

The code is below. You need to change the px value which is the original size and the first number in the first set of brackets.

p { font-size: calc(14px + (18 - 14) * ((100vw - 300px) / (1600 - 300))); }
