Flex grid

Grid system

Element uses Flexbox for its grid system. With recent compatibility issues being resolved, using Flexbox is an easy way to implement a responsive grid system with minimal code. It allows easy manipulation of child content like vertical centering that float based layouts struggle to offer without hacking away at workarounds.

Element heirarchy

Introduction

Element's Flexbox grid uses a container element (likely a section), a row and a child column. This is the standard layout.

<section class="container"> <div class="grid-row"> <div class="grid-col"> <h2>Hello world</h2> </div> </div> </section>

Container

The container is the parent element of the rows which spans the full width of the page. The rows are then nested within.

<section class="container"> <div class="grid-row"> <!-- Content here --> </div> </section>

Row

The row is the parent element of the columns. This is set to display:flex and a negative margin is applied to offset the margin applied by the columns

<div class="grid-row"> <!-- Columns here --> </div>

Columns

Columns are the child element of rows which hold your content. They are auto set to be equal width to fill your rows no matter how many columns are present or you can specify an exact width.

<div class="grid-row"> <div class="grid-col"> <!-- Content here --> </div> </div>

Multiple columns

You can nest multiple columns into a row and they will be automatically sized at an equal width. Gutters are applied as standard.

<div class="grid-row"> <div class="grid-col"> <!-- Content here --> </div> <div class="grid-col"> <!-- Content here --> </div> <div class="grid-col"> <!-- Content here --> </div> </div>

Aligning flex items

You can align individual flex items outside of the alignment of the parent element. For example you could push a button to the right of a column with the rest of the content being pushed to the left. Use the following classes:

.flex-item-right .flex-item-left
