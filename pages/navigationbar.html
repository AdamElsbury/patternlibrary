Navigation bar

Navigation bar

Element has a navigation bar template you can use. It is intentionally minimalistic as navigation bars vary greatly from website to website to allow you to add your own style. The navbar is responsive.

Using flexbox element allows you to position links to the left, center and right of the container div.

Basic usage

Container

The navbar has a container element .nav-container. Ensure your children links and logos go inside this container

<div class="nav-container"> <!-- Links and logos go here! --> </div>

Nav items

You can place an <img> element or a <ul> element within the nav container. There are two classes .nav-links and .nav-logo:

  • <img> Wrap in an <a> tag and use the class .nav-logo
  • <ul> Use the child class .nav-links
  • <li> Use the child class .nav-item

<a> links must be nested within <li> elements within the unordered list and given the .nav-item class.

<div class="nav-container"> <a href="#" class="nav-logo><img src="#"></a> <ul class="nav-links> <!-- Links go here --> </div>

Positioning child elements

You can position either type of element within the nav-container using an order class:

  • .item1
  • .item2
  • .item3

Item-1 is displayed first, item-2 is second and item-3 is last (going from left to right). Each element has an equal space between so you don't have to worry about positioning elements.

Sizing navigation bar

The way the navigation bar works is it automatically expands to whatever content is placed within it. If you wish to give the navigation bar a minimum height, please change the size of the tags using height and line height (please ensure both values are the same)

.nav-item a { height: (insert your value here in px); line-height: (insert your value here in px); }
