Jumbotron

Element has a pre-configured jumbotron. The jumbotron is intended to be a self contained element for a primary call to action on a page such as the example "Sign up to our newsletter".

Basic usage

HTML markup

Start with a .jumbotron container, add the standard grid elements and nest your content. Use the HTML code below if you wish to have a jumbotron like the example above.

<div class="hero"> <div class="container jumbotron"> <div class="grid-row padding"> <div class="grid-col"> <h3>Sign up to our newsletter</h3> <p>Your copy here!</p> <input type="text" class="margin-top" placeholder="Enter email address"> <button class="btn-primary">SIGN UP </button> </div> </div> </div> </div>

Adjust width of the jumbotron element

In order to adjust the width of the jumbotron section change the width selector in the CSS rule for .jumbotron. It is set to 70% as default.

.jumbotron { width: 300px; }
