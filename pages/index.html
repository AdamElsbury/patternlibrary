About Element

Introduction

Welcome to Element design system. EDS is a pattern library for usage by developers and designers to speed up their working process. Whether you are using this design system for prototyping, experimentation or for use in a live project you can ease some of the time needed to get up and running with our FlexGrid system or pull components directly for use in your web pages.

This design system was created by the owner Adam Elsbury to consolidate personal assets and organise things like text and heading sizes and weights for use in other projects instead of having to start from scratch. Hopefully it can help you too!

Why use Element?

Element is atomic

Following Brad Frost's Atomic design principles, Element was created with the developer or designer in mind. All components work in the same interface and group together without issue. No matter how big or small your project is. Modularity is our priority.

Ability doesn't matter

Whether you are putting together your first "Hello world!" template or submitting a finished website to a client - Element caters for all. Our literature is easy to understand for beginners and we have jargon buster sections for our less experienced users!

User centric design

Every single component has been rigorously user tested with live participants to ensure everything included is of the highest quality. It also gives you the peace of mind of not having to test it yourself. Each component is tested alone and in groups of elements to ensure design continuity.

Rapidly evolving

Element was started as a University project but will continue to grow to help designers and developers alike. If you have any feedback about anything that could be done better, or if you would like to see something added that is missing - get in touch!

Lightweight library

With less than 500 lines of CSS Element is a small, lightweight library that will not impact your page load times.

For developers

  • Plug fully responsive components into your webpage
  • Quickly edit the style of any component with SCSS variables
  • Built with as few lines of code as possible to enable fast load times
  • Transition effects built in to most components
  • Quickly build layouts with our pre built FlexGrid system

For designers

  • Use our style guide and UI asset library to rapidly create interfaces
  • All assets are user tested
  • Built with Adobe XD
