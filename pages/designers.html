For designers

Element vanilla library

The design library contains all the components of Element for you to use in your design software. The component sheet is available as an Adobe XD artboard and individually as SVG's in case you use Sketch or other software. Additionally we have also included a style guide.

Get the kit

1. Install Adobe XD

To access and use the full artboard of design assets you must have the latest version of Adobe XD installed.

2. Download the kit

To access and use the full artboard of design assets you can download the Adobe XD artboard below

3. Open the kit in Adobe XD

Save the file and either open directly or load it as a new project in Adobe XD

Start designing!

You're all good to go! The components are saved as symbols and will appear in the assets library on the left side of the screen. Be aware changing the size of any of the components will adjust the size wherever these assets are used.

If you want to adjust size or anything else copy the element and make a new asset.
