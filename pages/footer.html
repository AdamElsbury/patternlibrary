Social media

Footer

Element has a pre-configured footer to nest links, paragraphs and social media icons. A footer generally contains information about the company/service that is not deemed important enough to include in the main body of the webpage, and to provide links to things like the sitemap.

Basic usage

HTML markup

Start with a .footer container, include the column layout and place your required number of columns. Within the columns nest your content. The footer's CSS rules will do the rest. Include .footer-favicon as a class in the column that contains social media links. The footer will expand to the containing content.

<footer class="container footer"> <div class="grid-row padding-small"> <div class="grid-col"> <h4>Footer paragraph</h4> <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras vulputate felis a ultrices egestas. Nunc maximus pulvinar leo in blandit. Duis ullamcorper bibendum laoreet. Duis auctor, enim vel rhoncus condimentum, nulla sem placerat nulla, in accumsan turpis velit vel mauris. Aliquam pulvinar magna in metus iaculis, at finibus est volutpat.</p> </div>
<div class="grid-col text-center"> <h4>Footer links</h4> <ul> <li><a href="#">LINK 1</a></li> <li><a href="#">LINK 2</a></li> <li><a href="#">LINK 3</a></li> <li><a href="#">LINK 4</a></li> </ul> </div>
<div class="grid-col footer-favicon text-right"> <h4>Social media</h4> <a href="#"><i class="fab fa-facebook-square margin-right" style="font-size:3em;"></i></a> <a href="#"><i class="fab fa-facebook-square margin-right" style="font-size:3em;"></i></a> <a href="#"><i class="fab fa-facebook-square margin-right" style="font-size:3em;"></i></a> </div> </footer>
