Buttons

Buttons

Element comes with 6 different button styles aimed at usage in different scenarios based on content hierarchy or calls to action (CTA). The buttons can be found below.

Basic usage

The .btn class can be used on different elements such as:

  • <a> Anchor tags
  • <button> Button tags
  • <input type="submit"> Input submit tags

Variations

Primary

The primary button style is meant to be used as the main call to action on the page and is intended to be used highly in the content hierarchy. It is designed to stand out within the interface. The primary button comes with a hover effect to give the user feedback.

<button class="btn-primary">PRIMARY</button>

Secondary

The secondary button style is meant to be used as the secondary call to action on the page and to compliment the primary action. It is intended to indicate an important action but is not the primary user goal.

The secondary style is intentionally less apparent within the interface.

<button class="btn-secondary">SECONDARY</button>

Full width

The full width button style is generally used within mobile viewports or when you wish to have the button fill the entire width of its parent element in elements such as cards or navigation bars. The full width button comes with a hover effect to give the user feedback.

<button class="btn-primary">PRIMARY</button> <button class="btn-secondary">SECONDARY</button>

Rounded

You can apply the rounded modifier to either the primary or secondary button styles to change the default styling to increase the border radius. You can also apply it to the full width buttons.

<button class="btn-primary rounded">PRIMARY</button> <button class="btn-secondary rounded">SECONDARY</button> <button class="btn-primary fw rounded">SECONDARY</button> <button class="btn-secondary fw rounded">SECONDARY</button>

Square

You can apply the square modifier to either the primary or secondary button styles to change the default styling to reduce the border radius. You can also apply it to the full width buttons.

<button class="btn-primary square">PRIMARY</button> <button class="btn-secondary square">SECONDARY</button> <button class="btn-primary fw square">SECONDARY</button> <button class="btn-secondary fw square">SECONDARY</button>

No hover

You can apply the no hover modifier to any button element to remove the hover effect completely.

<button class="btn-primary no-hover">PRIMARY</button> <button class="btn-secondary no-hover">SECONDARY</button> <button class="btn-primary fw no-hover">SECONDARY</button> <button class="btn-secondary fw no-hover">SECONDARY</button>
